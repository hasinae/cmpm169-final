Description

What are you trying to accomplish with this experiment?

In this experiment, I aimed to create a chatbot that dynamically changes its tone—such as shifting to a poetic style—based on user input. The chatbot uses a grammar-based system to generate creative responses and incorporates dynamic visuals like background colors and shapes that change depending on the tone of the conversation. The goal was to combine technical elements like tone detection and response generation with creative visuals to create an engaging and interactive user experience.

Technical

Describe in detail what you did technically and how it works?

I built a chatbot using p5.js that detects the tone of user input (e.g., playful, poetic, or neutral) based on specific keywords. The bot uses predefined grammar templates to generate responses tailored to the detected tone. For example, if the user mentions words like "dream" or "heart," the bot shifts to a poetic tone and generates responses like, "In the realm of {topic}, dreams take flight."
